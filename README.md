# appbi

## Features
This program is made to help users Install packages easily with one click,
Made for developers, and users.

### For Users

* Clean UI, and Easy to use.
* Shows Suggetions For bundles, like, Office, Gaming, Art, and Video Edting.
* Real time loading with low network usage.
* Uses flatpak to load packages.

### For Developers

* Easy to design programs or upload custom code.
* Uses program languages like, lua, C++, and Python.
* Able to modify code later using Visual Studio Code, and Built in text editor.

## Build
